# Decision Trees

## Overview
This lab covers decision tree algorithms for both classification and regression tasks. Explores tree construction, pruning, and ensemble methods.

## Files
- **L04.ipynb**: Complete decision tree analysis and implementation notebook
- **Problem_Statement.md**: Problem statement and learning objectives

## Key Concepts
- Tree construction algorithms (ID3, C4.5, CART)
- Splitting criteria (Information Gain, Gini Index)
- Overfitting and pruning techniques
- Ensemble methods (Random Forests, Gradient Boosting)
